# PicoCTF Challenges

This repository contains several challenges built for **PicoCTF**.  

Each directory represents a challenge.  
To view a challenge description, navigate into its directory and open the `problem.md` file.

## Building Challenges
For information on how to build challenges, see:  
- [picoCTF/start-problem-dev](https://github.com/picoctf/start-problem-dev/)  
- [picoCTF/challenge-manager](https://github.com/picoCTF/challenge-manager)
